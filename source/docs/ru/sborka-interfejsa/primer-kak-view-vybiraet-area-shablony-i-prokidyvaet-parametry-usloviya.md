Внутри `template.php` представления области обычно хранится вызов грида с большим массивом параметров. Там встречаются типовые элементы:

* выбор шаблона area: `ROW_5_COL_0_AREA_0_TEMPLATE => "page.title"`  
* параметры конкретного area/блока: `ROW_5_COL_0_AREA_0__PAGE_TITLE__TITLE_MODIFIER => ""`  
* условный показ строки/колонки: `ROW_5_USE_CONDITION => "Y"` \+ `ROW_5_CONDITION_PROPERTY => "show_title"` \+ сравнение/значение

Это и есть “сохранённая конфигурация”, которую вы описали: визуальный редактор формирует/обновляет параметры и сохраняет их в `{site_dir}/simai.data/grid/view/.../template.php`.
