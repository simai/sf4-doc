У мастера есть директория (`WIZARD_DIR`) и конфигурационный файл (`.wizard.config.php`). Внутри директории мастера обычно хранится папка `data/` — это “пакет” (архивы, конфиги, шаблоны), который мастер может собирать и распаковывать.

Типовой состав `WIZARD_DIR/data/` (по смыслу):

* `data/config/` — файлы конфигов (типы инфоблоков, опции, группы пользователей и т.д.)  
* `data/iblock/` — архивы инфоблоков  
* `data/component/` — архивы компонентов  
* `data/module/` — архивы модулей решения  
* `data/public/` — архивы публичных файлов сайта (в т.ч. по SITE\_ID)  
* `data/template/` — архив шаблона  
* `data/php_interface/` — добавки для окружения (например, файлы-строки для dbconn/init)

Это важно для документации: “пакет” — не абстракция, а конкретная структура, с которой удобно работать при переносах и обновлениях.
