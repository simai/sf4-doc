# Track B — Развитие ядра SF4

Фокус: устройство модулей, внутренние подсистемы, публичные точки расширения, обновления и совместимость.

Навигация:
- `modules.md` — экосистема модулей
- `extension-points.md` — публичные точки расширения
- `classes.md` — ключевые классы и подсистемы
- `constants.md` — константы и базовые пути
- `conventions.md` — конвенции и код-стайл
- `assets.md` — ассеты ядра
- `update-compat.md` — обновления и совместимость
- `export-import.md` — экспорт/импорт и миграции
- `search.md` — интеграция с поиском Bitrix
- `security-permissions.md` — права доступа и безопасность
- `api-examples.md` — примеры API (настройки, iblock-хелперы, ассеты)
