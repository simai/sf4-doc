# Траблшутинг (Track A)

## Назначение
Фиксирует подтверждённые проблемы и способы их диагностики для решений на SF4.

## Где применяется
Работа с `{site_dir}/simai.data`, wizard actions `/simai/wizard/action`, подключение ассетов и модулей.

## Где находится
- Каталоги данных и конфигов: `source/structure.json`, `source/sf4-structure.enriched.json`.
- Источники Q&A: `source/SIMAI-SF4 Часто задаваемые вопросы.md`, `source/SIMAI-SF4 Курс разработчика.md`.

## Требования
Использовать только подтверждённые симптомы/решения из локальных файлов; не добавлять предположения.

## Как это работает
Не найдено в локальных данных. Искомое: регламент диагностики SF4 (логирование, чек-листы). Искали в `source/file-functional.md`, `source/file-frontend.md`.

## Настройка и параметры
Не найдено в локальных данных. Искомое: параметры включения/отключения диагностики. Искали в `source/modules`, `source/sf4-structure.enriched.json`.

## Примеры из репозитория
Источник: `source/SIMAI-SF4 Часто задаваемые вопросы.md`
```
2. **Кеш сайта не очищается**. Кеш сайта используется для ускорения работы сайта и содержит кешированные данные информации сайта. После обновления информации кеш обычно очищается от старых данных и добавляет новые. В редких случаях происходит сбой, и старые данные не удаляются. Вы можете попробовать удалить кеш в ручную.
```

## Типичные ошибки и диагностика
- Изменения в `{site_dir}/simai.data` не применяются → очистить кеш, проверить права на запись.
- 404 на wizard actions → проверить наличие файлов в `/simai/wizard/action` и активность модуля `simai.framework`.
- Ассеты не подхватываются → сверить `{site_dir}/simai.data/config/.asset.config.php`, отсутствие дублей библиотек.
- Формы настроек не открываются → убедиться, что `simai.property` установлен.

## Что должен уметь читатель после
- Проверять основные точки отказа: кеш, права на `{site_dir}/simai.data`, наличие wizard actions.
- Использовать FAQ/курс разработчика для подтверждённых решений.
- Фиксировать отсутствие нужного ответа и запросить добавление в `source/*`.
