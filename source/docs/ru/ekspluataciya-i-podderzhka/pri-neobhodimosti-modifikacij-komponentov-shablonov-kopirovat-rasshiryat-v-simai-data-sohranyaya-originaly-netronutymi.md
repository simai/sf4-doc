Правильная логика доработок в SF4 — не “редактировать существующее”, а **добавлять свой слой поверх**:

* нужно добавить область → создаём папку и `template.php` в `simai.data/template/area/...`, подключаем через `IncludeArea::includeTemplateArea('...')`;  
* нужно изменить внешний вид/поведение блоков → делаем override через `simai.data/grid/...`;  
* нужно добавить кнопки/панели/обвязку → используем проектные файлы в `simai.data/template/...`.

Этот подход делает обновления предсказуемыми: системный слой обновился, проектный слой остался.
