| Code | Назначение | Реализация | Ключевые параметры | Примечания |
| :---- | :---- | ----: | :---- | :---- |
| `dir.make` | Создание папок | `action.php` | `parameter` — массив путей (`/path/dir/`) | Создаёт директории в пределах сайта (от docroot). |
| `file.copy` | Копирование файлов/папок | `action.php` | `parameter[]: source, destination, rewrite` | `parameter` — массив операций копирования. |
| `file.delete` | Удаление файлов/папок | `action.php` | `parameter` — массив путей (обычно начиная с `/`) | Удаляет через `DeleteDirFilesEx`. |
| `file.rename` | Переименование/перенос | `action.php` | `parameter[]: source, destination` | Поддерживаются подстановки `#KEY#` из входных данных. Шаг “порционно” идёт по списку. |
| `file.create` | Создание файла с текстом | `action.php` | **Требует уточнения** (ключи параметров не извлечены) | В локальных данных есть обработчик, но без уверенного списка ключей. |
| `file.add` | Добавление PHP-кода в файл | только описание | — | В локальных данных нет исполняемого кода. |
| `file.encode.win1251` | Перекодировка lang-файлов в win-1251 | только описание | — | По вашему комментарию не нужно (Bitrix ушёл от этой кодировки). |
| `file.zip` | Упаковка в архив | только описание | — | В локальных данных нет исполняемого кода. |
| `file.unzip` | Распаковка архива | только описание | — | В локальных данных нет исполняемого кода. |
| `cut.names` | Массовое переименование файлов | только описание | — | В локальных данных нет исполняемого кода. |
| `restore.names` | Откат массового переименования | только описание | — | В локальных данных нет исполняемого кода. |
