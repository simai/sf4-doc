| Code | Назначение | Реализация | Ключевые параметры | Примечания |
| :---- | :---- | ----: | :---- | :---- |
| `data.add.config` | “Перевод параметров в конфигурацию” (по сути — отдать `parameter` наружу) | `action.php` | `parameter` (любая структура) | Фактически кладёт `parameter` в `data_output_code`. Удобно как “шлюз” между этапами. |
| `data.add.property` | Форма ввода от пользователя | только описание | — | Используется как логический шаг ввода, но **исполняемый код не найден**. Нужна фиксация, где именно рендерится форма в вашем мастере. |

| `data.export.file` | Сохранение массива данных в файл | `action.php` | `file` — путь файла назначения | Сохраняет данные шага в файл через внутренний helper сохранения. |
